What is Traceroute?

Traceroute is a network diagnostic tool that helps us understand the path packets take from the source to the destination.

Understanding the Output

Hop: Router number (1, 2, 3...)

IP Address: IP address of the router

Hostname: Router's domain name (if available)

Location: Physical location of the router (city, country)

RTT: Round-Trip Time (latency in milliseconds)

* (Star): Router did not respond (timeout/firewall)
